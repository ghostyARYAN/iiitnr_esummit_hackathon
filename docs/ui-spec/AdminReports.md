**Source File**: [AdminReports.tsx](file:///d:/ARyan/iiitnr-esummit/hackathon/src/pages/AdminReports.tsx)

## UI Structure

### Sector Breakdown Table
- Summary of application statistics per sector.
- **Columns**:
  - Sector Name
  - Total Applications
  - Pending Applications
  - Finalized Applications

### Trend Chart
- Bar chart showing the number of application submissions per month for the last year.

## Functionality
- **Data Aggregation**: Summarizes application data across the entire platform.
- **CSV Export**: Provides a downloadable report of sector-wise performance metrics.
- **Role-Based Access**: Accessible only to users with the "admin" role.
- **Real-time Analytics**: Reflects application trends and sector-specific metrics.
